## 스터디 정보
* 시작 일자: 2023.01.08 ~
* 진행 시각: 매주 일요일 오전 10시
* 모임 방식: 
  * zoom을 통한 온라인 참여가 기본입니다.
  * 한 달에 한 번 오프라인 모임을 가질 예정입니다.
* 진행 방식:
  * 매주 한 명씩 `스파크 완벽 가이드` 내용을 발제합니다.
  * 동시에 일주일 동안 발제 내용 관련 실습을 진행합니다.
  * 발제 이후에는 실습 내용 관련 트러블 슈팅 등을 논의합니다.
